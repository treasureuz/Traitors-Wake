# Traitor's Wake
Description: Grid based memory game; Pirate themed.
- Different paths/lanes/lines created that the user must 1:1 recreate to pass the level
- The path is highlighted after each play
- The user is only given a few seconds to memorize the highlighted pth
- You start bottom left and finish top right
- The paths change, and the grid width and height changes based on difficulty

Story: A fellow pirate betrayed your pirate ship, The Square Sail, by relaying confidential information to nearby rival pirate ships. With a bounty on his head, the traitor escaped The Square Sail and is headed on to the main land. Your job as the captain is to chase this traitor by memorizing and following his trail on the sea. As you gain momentum, his routes gets trickier and he start throwing rocks to slow you down. Do not worry! Collecting nearby chests grants upgrades that get you closer, leading to his demise.
